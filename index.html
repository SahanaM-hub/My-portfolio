ğŸ“‚ Tech Glossary

HTML
HTML is the standard markup language used to create and structure the content of documents displayed in a web browser, making it the most basic building block of the World Wide Web.
CSS
The acronym CSS stands for Cascading Style Sheets. It is a style sheet language used to describe the presentation and styling of a document written in a markup language, most commonly HTML. While HTML provides the structure and content (the skeleton), CSS provides the design (the clothes and decorations).
JS
JavaScript (JS) is a high-level, interpreted programming language that is one of the three core technologies of the World Wide Web, alongside HTML (structure) and CSS (style). While HTML and CSS define the what and how a web page looks, JavaScript defines the behavior and interactivity of the page.
